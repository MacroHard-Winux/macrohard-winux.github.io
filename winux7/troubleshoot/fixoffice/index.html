How to Fix Microsoft Powerpoint/Onenote Not Starting/Crashing on Start



  1. Step 1

    Press up the Alt+F2 keys on your keybaord together and a run menu will show up. It is bit like the Windows run window when you press Windows+R key on Windows 7

  2. Step 2

    Enter the command "winecfg" into the text box, and press Enter. A new window should popup

  3. Step 3

    This is the configuration window of the Wine Compability Layer used to run Windows Programes on Winux7.

  4. Step 4

    Click on the "libraries" tab. to configure the wine libraries

  5. Step 5

    From the drop down menu find a select the "ricked20" option.

  6. Step 6

    Click on the "Add" button on the right of the drop-down menu to add it into our existing overrides.

  7. Step 7

    Click on the Edit button under the "Add button to to edit our override".

  8. Step 8

    Change the load order to Native (Windows). This makes all the apps in this wineprefix use the 'riched20' library that has been installed by MS-Office instead of the default wine version.

  9. Step 9

    Click on the "OK" button to close the window.

  10. Step 10

    Click the "Apply" button the apply the changes that we just made.



    Note: Dont apply the changes before you install Microsoft Office otherwise you will not be able to enter the product key into the text box during setup

  11. Step 11

    Click on "OK" to close the Wine configuration window, and Microsoft PowerPoint and Onenote should be working again. :)
